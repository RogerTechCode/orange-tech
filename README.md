# orange-tech
Orange Futebol Clube — Site Institucional
Sobre o Projeto

O Orange Futebol Clube é um site institucional desenvolvido para representar um clube de futebol fictício, com foco em organização de conteúdo, identidade visual moderna e simulação de um ambiente real de produção.

O projeto inclui um sistema completo com área pública e painel administrativo, permitindo gerenciamento dinâmico de informações como notícias, patrocinadores e equipe.

Funcionalidades
Site Público
Página inicial com identidade visual moderna
Seção de história do clube
Listagem dinâmica de notícias
Exibição de patrocinadores
Seção de equipe (diretoria e comissão técnica)
Layout responsivo para diferentes dispositivos
Painel Administrativo
Autenticação de administrador
Cadastro de notícias com mídia (imagem ou vídeo)
Edição e exclusão de notícias
Cadastro de patrocinadores
Gerenciamento de equipe com dados persistidos

Como Executar o Projeto
Clone o repositório:
git clone https://github.com/RogerTechCode/orange-tech.git
Acesse a pasta do projeto:
cd orange-tech
Abra o arquivo index.html no navegador
ou utilize uma extensão como Live Server no Visual Studio Code

Integração com Banco de Dados

O projeto utiliza o Firebase Firestore para armazenamento de dados dinâmicos.

Principais coleções utilizadas:

news
sponsors
managers

Observação: é necessário configurar suas próprias credenciais do Firebase para funcionamento completo.
